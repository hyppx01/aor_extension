# Changelog

All notable changes to AI Output Clipper will be documented in this file.

## [1.0.0] - 2024-12-30

### Added
- Initial release
- Manual content input with title and markdown content
- GitHub API integration for publishing posts
- Chinese character support (UTF-8 to Base64 encoding)
- Chrome storage for GitHub configuration
- Configurable target path
- Optional date prefix in filenames
- Connection testing feature
- Character counter
- Dark mode support
- Status indicators
- Preview filename feature

### Security
- Token stored securely using Chrome storage API
- Only communicates with GitHub API
